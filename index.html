My TechStack

HTML

The dependency of the web made me love HTML, it was the first programming language(technically a markup language) I learned. Writing HTML alone without CSS for a while, I’ve learnt to write HTML semantically, making it understandable by accessibility tools like text to speech and screen readers.

CSS

Growing up artistically limited, CSS gave me all the resources and freedom I needed to be artistic. Currently, I write CSS for fun. As a matter of fact, anytime I’m writing in a scripting language and I encounter a developer’s block or a bug that I’m too mentally strained to solve, I write more CSS to clear my mind. I also enjoy writing CSS superset, SASS.

JavaScript

I’ve always loved JavaScript, I remember when I saw “The Light Bulb” on W3School’s JavaScript page I was surprised, My exact reaction was “Holy [bleep] JavaScript is magical”. Then it dawned on me that HTML can be compared to drawing on a sheet of paper, CSS to painting that drawing and JavaScript to breathing life into that drawing. Pushing the limits of what I can do with JavaScript has been my motivation while learning the language, and hopefully, It’ll always be.

React

When I found out that instead of manipulating the browser's DOM directly, React creates a virtual DOM in memory, where it does all the necessary manipulating, before making the changes in the browser DOM, I got interested in the Library. I currently have beginner experience in using the library, my first project was emulating the actions of a shopping cart in an e-commerce website and updating shopping cart without refreshing the DOM.
